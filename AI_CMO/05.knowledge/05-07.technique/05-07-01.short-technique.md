### **第3部：【エンゲージメント戦術編】 視聴者を1秒たりとも離脱させない技術**
*ショート動画特有の視聴行動に対応し、冒頭で惹きつけ、最後まで飽きさせないための戦術。*

#### **3-1. 超初動インパクト（最初の1秒）**
| テクニック名 | 概要 (Description) | 具体例 (Example) |
| :--- | :--- | :--- |
| **注意喚起ワード** | 「見て！」「やばい」「えっ」といった反射的に注意を引く単語から始める。 | ちょ、やばいやばい (管理番号: 014) |
| **“自分ごと化”トリガーワード** | 「〇〇な人」「〇〇してる人」と呼びかけ、自分に関係があるか瞬時に判断させる。 | 2年以上同じ車に乗ってる人おめでとうございます (管理番号: 005) |
| **逆説・否定フック** | 「〇〇やめて」「まだ〇〇してるの？」と常識を否定し、「なぜ？」という疑問を抱かせる。 | ラクトロン買おうとしてる人一回ストップ (管理番号: 046) |
| **“秘密の暴露”フック** | 「実は」「ここだけの話」で始め、価値のある情報が聞けることを示唆する。 | 実は自宅で簡単おなら便流し (管理番号: 029) |
| **視覚的・聴覚的ノイズ（※演出要素）** | （注釈）台本外の要素だが、冒頭のインパクトのある効果音や映像で、感覚に直接訴えかける。 | 冒頭の驚いた表情や、効果音など。 |

#### **3-2. 中間離脱防止**
| テクニック名 | 概要 (Description) | 具体例 (Example) |
| :--- | :--- | :--- |
| **マイクロ・クリフハンガー** | 「なんで？」「どういうこと？」と小さな疑問を意図的に作り、答えを少し先に配置する。 | え、なんで返済なくなったの? → この前ニュースでさ... (管理番号: 006) |
| **情報の小出し（チャンキング）** | 「1つ目は…」「さらに…」と情報を分割してテンポ良く提示し、次への期待感を維持する。 | 1つ目はストップニキビ...2つ目は肌保湿の相棒...3つ目は飲むニキビ治療... (管理番号: 001) |
| **ベネフィットの“予告”** | 「異次元の手触りが待ってた」のように衝撃的な結末を先に予告し、プロセスへの興味を引く。 | あれ試したら弁ドバドバ出てトイレ壊れた (管理番号: 079) |
| **共感と納得のループ** | 視聴者の「あるある（共感）」と商品の「なるほど（納得）」を交互に繰り返し、一体感を生む。 | （共感）毎日飲むペットボトル...超ストレス → （納得）それを解消してくれるのがウォーターサーバー... (管理番号: 016) |

### **第4部：【表現と行動喚起編】 心を動かし、「今すぐ」行動させる技術**
*メッセージの効果を最大化する「伝え方」と、購入への最後の一押し。*

#### **4-1. デリバリー技術（伝え方）**
| テクニック名 | 概要 (Description) | 具体例 (Example) |
| :--- | :--- | :--- |
| **友人トーク口調** | 「〜じゃん」「マジで」といったタメ口を使い、友人からのおすすめのような親近感を演出する。 | 美容院とかカラコンより安いのやばくない? (管理番号: 062) |
| **独り言・心の声の投影** | 「正直後悔してます」のように本音をさらけ出し、リアルさと信頼性を高める。 | なんでもっと早く飲み始めなかったのかなーって今は後悔してるの (管理番号: 017) |
| **“あなた”へのパーソナライズ** | 二人称（あなた）を多用し、視聴者一人ひとりに語りかけている感覚を生み出す。 | あなたも実はそうしてませんか (管理番号: 009) |
| **権威のカジュアルダウン** | 権威性を、友人から聞く噂話のようなカジュアルな言葉で伝え、信頼性と親近感を両立させる。 | めぐみも言ってるんでガチなやつですよ (管理番号: 025) |
| **一人称と三人称の自在なスイッチ** | 自分の体験（一人称）と客観的な情報（三人称）をシームレスに行き来し、共感と納得感を両立させる。 | （一人称）私は定価で...買ったけど、（三人称）48%オフで買えるキャンペーンやってた (管理番号: 007) |

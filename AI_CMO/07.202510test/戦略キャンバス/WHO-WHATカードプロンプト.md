## 役割
あなたは、ダイレクトマーケティングを専門とする優秀な戦略プランナーです。特に、WEB広告における新規顧客獲得の最大化（CPAの抑制と獲得件数の増加）を得意としています。

## 最終ゴール
これからインプットされる情報を基に、顧客インサイト（WHO）と提供価値（WHAT）の最適な組み合わせを「WHO-WHATカード」として、広告クリエイティブのアイデアの元となる仮説を可能な限り多く、多様な切り口で立案することが最終ゴールです。

## 指示
以下の「入力データ」を深く分析し、「出力フォーマット」に厳密に従って、「顧客インサイト」「商品の特徴」「得られるメリット」の3つの要素で構成されるWHO-WHATカードを、最低20個以上生成してください。

---

## 入力データ

### 商品情報のサマリー
【ここに、これまで作成したような商品の特徴・メリット・実績・権威性・オファーに関するサマリーを貼り付けてください】

### 顧客の一次情報・行動データ
【ここに、顧客インタビュー、アンケート結果、レビューサイトの口コミ、SNSでのUGC、過去の広告運用データ、ウェブサイトのアクセス解析データなど、顧客の「生の声」や「実際の行動」に関する情報を貼り付けてください。情報がない場合は「特になし」と記載してください】

---

## 出力フォーマット

以下のMarkdown形式の「WHO-WHATカード」のフォーマットに厳密に従い、「顧客インサイト(WHO)」「商品の特徴」「得られるメリット」の3点セットを具体的に記述してください。それぞれの内容は、誰が読んでも理解できるシンプルな言葉で表現してください。

### WHO-WHATカード #（番号）

| 項目 | 内容 |
| :--- | :--- |
| **顧客インサイト(WHO)** | （ターゲットが抱える具体的な悩み、欲求、判断基準などを記述） |
| **商品の特徴** | （上記のインサイトに応える、具体的な商品の事実・仕様を記述） |
| **得られるメリット** | （特徴によってもたらされる、顧客にとっての具体的な便益・理想の未来を記述） |

---

**【記述例：マウスピース矯正の場合】**

### WHO-WHATカード #1

| 項目 | 内容 |
| :--- | :--- |
| **顧客インサイト(WHO)** | 100万円以上かかる歯列矯正は高すぎる。 |
| **商品の特徴** | 月々3,000円台から始められる料金プラン。 |
| **得られるメリット** | お金をかけずに歯並びをきれいにできる。 |

### WHO-WHATカード #2

| 項目 | 内容 |
| :--- | :--- |
| **顧客インサイト(WHO)** | ワイヤー矯正は見た目が目立つから嫌だ。 |
| **商品の特徴** | 透明で目立ちにくいマウスピースを使用。 |
| **得られるメリット** | 周りの人に気づかれずに矯正できる。 |

## 制約条件
- 必ず20個以上のWHO-WHATカードを生成してください。
- 生成するカードは、顧客の多様な悩みや欲求、商品の様々な特徴を反映した、多角的な視点を持つものにしてください。
- 「顧客インサイト」「商品の特徴」「得られるメリット」の3つの要素は、必ず明確な因果関係・対応関係になるように記述してください。
- 直ちに行動を開始してください。
